# NO MONEY. NO HAPPY.

一个给没钱的人看的网站。

## 设计理念

这不是一个投资官网、白皮书站或产品介绍页。这是一个**情绪出口 + 身份确认页面**。

用户打开网站的第一感觉应该是：**"他懂我。"**

## 风格特点

- **Brutal** - 残酷、直接
- **Honest** - 诚实、不掩饰
- **Meme** - 像会被截图转发的表情包
- **Poor-core** - 穷人美学
- **Pixel / Retro** - 像素风格、复古

## 技术实现

- 纯 HTML + CSS
- 像素字体：Press Start 2P
- 单页设计，无复杂交互
- 响应式布局

## 功能特性

### ✨ 新增功能

1. **Logo展示**
   - 浏览器标签页favicon
   - 顶部导航栏logo
   - Hero区域大logo（带发光动画）

2. **中英文切换**
   - 右上角语言切换按钮（EN/中文）
   - 自动保存语言偏好
   - 全站内容实时切换

3. **推特链接**
   - 右上角推特图标
   - 悬停动画效果

4. **丰富的动画效果**
   - **Glitch故障效果**：文字闪烁、错位
   - **露宿街头风格**：街灯闪烁、随机抖动
   - **发光脉冲**：Logo和标题发光动画
   - **文字抖动**：像素风格抖动效果
   - **鼠标交互**：鼠标悬停文字跟随效果
   - **滚动动画**：滚动时的淡入淡出

5. **视觉增强**
   - 多层文字阴影和发光
   - 像素化渲染
   - 高对比度配色
   - 响应式设计

## 设计原则

> Design this site like a meme, not like a startup.
> 
> It should feel poor, honest, and emotional — not professional.

### 动画风格

- **露宿街头感**：闪烁的街灯、随机故障效果
- **Brutal美学**：强烈的抖动、故障、闪烁
- **情绪压迫**：大字体、强对比、动态效果
